Learning Git with neACM

In this workshop I have learned to use Git at a beginner to intermediate level. I completed the following steps:

  • Create an empty repository
  • Understand what is a working tree and the index
  • Stage and unstage changes to/form the index
  • View the repository history using the log command
  • Manage branches and resolve merges conflicts
  • Work with remote repository storage using ssh keys
